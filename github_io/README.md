# Docs with mkdocs

This folder contains the https://allartk.github.io/leaflet.offline/ website 

It is build with [mkdocs.org](https://www.mkdocs.org).

```shell
# create your venv eg:
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
# or to build
mkdocs build
```
